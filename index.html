How My Interest In I.T Started

Information technology first intrigued me in 1998 the day the we got our first computer. Months later, while waiting 45 minutes for an image loading in photoshop, I remember taking the side panel off the computer case. While looking at all of the hardware, I remember thinking “how does all of this work?” We didn’t have the Internet at that time so I couldn’t look it up and no one I asked seemed to know how it worked. That was about the time that I started taking apart every electronic device that wasn’t bolted down to the ground in an attempt to find out.

At about the same time our first computer arrived in our house I received a Lego Mindstorms robotics discovery kit as a gift. The robotics kit had light sensors, touch sensors and motors. It was possible to program your robot to move in different directions. It was then possible to program your robot to react to touch and light input with different motor outputs. This was life changing to me to see how this computer could react to the outside world through input output commands, it gave me my first piece of understanding of what was going on inside computers.

My Project

My project idea is a weather proof security camera that turns on when a motion sensor is triggered capturing video, it uses an Infared light to see in the dark. The security camera will be powered by an external power bank (like the ones commonly used to charge mobile phones).
It will also be possible to connect the camera to Wi-Fi so that the camera can interact with a phone application. The phone application would give you the ability to view the live image from the camera as well as previously taken footage and battery status. It would even be possible to move the camera with app control.

Motivation

A motion triggered security camera would have many applications, also it would appeal to many target markets, the first three that come to mind are the home security industry.
Budget directs home invasion statistics suggest that in 2017 there were 225900 recorded burglaries in Australia, that’s one burglary every three minutes this product could give piece of mind to those who have been previously victimized. (budget direct.com).
There were more than 300000 births in Australia in 2019 it could appeal to new parents that could use their baby monitor once their child is grown as a home security camera, (macrotrends.net)
According to rspca.org.au there are over 29 million pets in Australia alone. Many people want to know what trouble their cat / dog’s / fish gets up to when they are not around.

Description

When the security cameras motion sensor is triggered by an object moving within a certain radius, the camera will record video and turn on its infrared lights. Objects will still be visible to the camera in no and low light stations, because Infared LEDs produce light that a camera can pick up but our eyes cannot detect it will be difficult for the subject being recorded to know they are being recorded.
It will be possible to use the camera both inside and outside because of its waterproof shell and waterproof seal protecting its electrical components. It is also easily mountable and will come with mounting brackets both permanent fixtures and semi-permanent. The camera would to sit on a flat surface without the need for mounting at all.
The security camera would have the flexibility of being powered by an external battery housed within the waterproof shell meaning that you would not need to dismount the whole camera for charging this would be especially convenient if you had multiple cameras.

The camera itself would interact with Wi-Fi, it would communicate with a software application that would be compatible with windows, Linux, macOS, as well as android and iPhone. Using the Rasbery pi as a server it would then be possible to view the live feed from the camera or download and view previous videos that have been stored. It would also be possible to automatically send videos to other computers to store directly on their hard-drives through the Wi-Fi connection incase the camera itself was destroyed.
The software application would display the battery percentage, also have simple functions such as moving the camera up and down left and right.

Hardware/Software

Raspberry pi 4: a raspberry pi is a cheap small computer that could be used as a desk top computer, more importantly the raspberry pi has I/O (input / output) pins that are used to control different types of sensors, motors or anything that can run off of electricity.
The raspberry pi will control the input sensors such as the motion sensor and the output features like the infrared lights, servo motors and raspberry pi camera. It would also be used as a server to access files on the raspberry pi remotely by app.

An Sd card is a convenient way of storing data it will be used to store recordings, hold the raspberry pi’s operating system and coding.

Servo motors are a type of cheap small motors, they will be used to position/move the camera up, down and left and right.

Infared LEDs are LEDs that emit light with a higher wave length then our own eyes can perceive they are for or the camera to see in the dark.

Motion sensors pick up Infared light emitted from our body heat they are used to detect objects close by.

External power banks will be the battery for powering the whole project.

Wire is used transport electricity and signals.

A raspberry pi camera is a small cheap camera that turns on when you send a small voltage to it.

Mounting brackets: To keep the camera suspended where we want it.

A cat, for testing purposes and cuteness.

Visual studio is an IDE used with the .net open source development platform, with it you can build websites and software for Linux, Microsoft and macOS using C# & F#

Xamarin is a .net open source platform used to build native android and iOS apps using C# & F#.

Skills Required

The project would require several pieces of software to be written, firstly it would require the input/output configuration for the motion sensor, LED’s and raspberry pi camera to be written in python on the Rasbery pi OS. Knowledge of the raspberry pi command line interface would also be advantageous.
Another python program would be written to take the video from the raspberry pi and put it in an accessible place on the Internet for the cross-platform application to access live and download previous videos.
Another python program would be written to accept commands from the cross-platform software application. Giving it the ability to move the camera up down.
There is also need for the cross-platform software application itself and it would need to run on Linux, windows and mac OS. It would be created using visual studio, .net and c#.
Another cross-platform software application for android and iOS would also need to be created also using Visual studio and Xamarin.

This project would give people a cheap effective way of seeing what their cat gets up to when they are not home. It could also help prosecute intruders, thieves, and keep people's belongings safe when they are not there to do it themselves.
